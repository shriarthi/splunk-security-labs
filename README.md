# splunk-security-labs
SOC Analyst practice labs using Splunk, created while learning SIEM through TryHackMe and other platforms.

## About This Repo

This repository contains my hands-on Splunk projects completed while learning Blue Team and SOC Analyst concepts. These labs were inspired by learning paths from TryHackMe, Security Blue Team, and other SOC training platforms.

## Included Skills

-  Log collection & ingestion
-  Searching with SPL (Search Processing Language)
-  Creating dashboards
-  Correlation searches
-  Investigating security incidents
-  Detecting brute-force attacks
-  Analyzing Windows logs
-  Investigating web server logs
-  Detecting malware indicators
-  Creating alerts
